OPEN
OP.
プログラム マニュアル
-----------------------------------------------------------
[機能]
データの入出力を開始します

[書式]
OPEN"形式:[ファイル名]"[FOR OUTPUT][FOR INPUT][FOR APPEND]

[説明]
形式が COM の場合は SIO 入出力をします。
PC-G850 シリーズは COM1・PIO も指定できます。
　COM  = 半二重通信
　COM1 = 全二重通信
　PIO  = 8ビット入出力指定

形式が CAS の場合はカセットテープの入出力。
FOR OUTPUT で出力、FOR INPUT で入力になります。
ファイル名を指定した場合、FOR OUTPUT の場合はファイル名も含めて記録します、
FOR INPUT の場合はファイル名を検索して読み込みます。

E: の場合は RAM データファイルを入出力します。G850 で有効です。
ファイル名を指定します。
FOR OUTPUT で読込、FOR INPUT で書込、FOR APPEND で追加書込となります。

形式が LPRT の場合はミニ I/O の入出力をします。

すでに一つオープンされている場合に、更に OPEN しようとした場合は エラー 86 となります。
E: は2つまでオープンする事ができます。
